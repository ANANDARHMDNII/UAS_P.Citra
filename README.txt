
YOLOv12 (YOLOv8) Object Detection WebApp

Cara Menjalankan:
1. Install dependency:
   pip install -r requirements.txt

2. Jalankan aplikasi:
   streamlit run app.py

3. Upload gambar (misalnya botol.jpg) untuk melihat hasil deteksi.

Note: Model 'yolov8n.pt' akan otomatis di-download oleh library Ultralytics saat pertama kali dijalankan.
